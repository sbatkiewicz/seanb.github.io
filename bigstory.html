Emaily - Feedback App


Feedback?

Emaily is an app that any service-provider can use to receive feedback regarding their service by logging in and purchasing credits securely on my site, and filling out a simple survey that includes an email list of their clients. The app then sends an email to each recipient and collects the data from the client about how they answered the survey emailed to them. These results are then easily accessible to the service-provider on my site in their dashboard.

Try it here!

Technologies Used:

  • Used Node.js with Express to create route handlers
  • Designed React app frontend
  • Google oauth flow for secure login
  • Implemented Strip API for secure payments
  • Automate email process with SendGrid API
  • Connected app to MongoDB to store user data
