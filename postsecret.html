PostSecret


Can you keep a secret?

PostSecret is a site that lets the user anonymously post a secret they have to help them get it off their chest and feel better about it. Any user can view the secrets, but it does not show the user who posted each secret.

Technologies Used:

  • Used Node.js with Express to create route handlers
  • Google oauth flow for secure login
  • EJS templates
  • Connected app to MongoDB to store user data
