Rusty Sprite Player


Did you hear?

The Rusty Sprite Player was a fun project that made listening to music as a group easy! The music player works in conjunction with the Discord app. By communicating with a Discord Bot, the app allows any of the users to search Youtube's API and add any song to a queue that will play in our voice chat for all to enjoy together!

Technologies Used:

  • Used Node.js with Express to create route handlers
  • Designed React app frontend
  • Queried Youtube API to search for songs
  • Discord.js to build Discord Bot to play music
